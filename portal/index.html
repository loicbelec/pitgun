Modular Modules

Composable Rust crates. Run them standalone or weave them into a mesh.

1
01

Traffic Gateway

High-throughput ingress layer with HTTP/WebSocket endpoints (UDP, MQTT coming soon). Normalizes disparate wire formats into a unified TelemetryFrame, buffering and routing to the mesh. Use it standalone as a protocol bridge, or as the mesh entry point.

2
02

Core Engine

The brain of the mesh. pitgun-core provides the AST-based Formula Engine (Power = Torque * RPM), manifest-driven pipelines, and type converters. Embed it directly into your Rust application for local formula evaluation without any network overhead.

3
03

WASM Solver

Distributed compute orchestration. pitgun-solver compiles to WebAssembly for browser-based volunteer computing. Feed it a RiskAnalysisRequest and it executes Monte Carlo simulations in parallel, aggregating results for strategy optimization.

4
04

Policy Authority WIP

Governance layer for secure distributed systems. Designed to sign configuration manifests, enforce Tuning Policy Limits, and provide cryptographic proof that result A came from Config B. Foundational work underway.
